# student-management-system
A beginner-to-intermediate Python project demonstrating Object-Oriented Programming through a student management system with grading and ranking features.
## Features
- Create student objects
- Update scores
- Automatic grade calculation
- Find best studen
- Get top three students
## Concepts Used
- Object-Oriented Programming (OOP)
- Properties (Property)
- Sorting with lambda functions
## How to Run 
Run the main file:

python.main.py
